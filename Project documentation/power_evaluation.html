power_evaluation.compare_power(x1, x2)[source]¶

Compares power between x1 (int) and x2(int)

Parameters:

  • x1 – Strength parameter for first instance

  • x2 – Strength parameter for second instance

Returns:

(float) ratio of strength. returns 1 -> win 100%. Returns -10 -> win 1%
