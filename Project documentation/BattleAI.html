BattleAI.debunkObstacles(current_position, state, obstacles, hex_img)[source]¶

Function removing false positives from detected obstacles It works by moving mouse to current unit and pressing ctrl to display it’s possible moves as dark hexagrams this allows us to make good threshold to get the tiles we can really move on if our env assumes one of this tiles is an obstacles we correct that mistake

Parameters:

  • current_position – tuple - current position of our unit

  • state – 2D array of objects - current state of our environment

  • obstacles – list of instances of class Obstacle

  • hex_img – np.array - mask used in checking the possible moves

Returns:

state: 2D array of objects - current state of our environment, obstacles: list of instances of class Obstacle
