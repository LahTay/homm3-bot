calculate_movments_points_at_start_day module¶

calculate_movments_points_at_start_day.calculate_movments_points_at_start_day(hero)[source]¶

Function which calculate points of movments at start a day. It check how is the lowest speed unit in army, what artifacts and skills we have. On this basis it calculate bonus next add it to variable base_movment. :param hero: Hero class :return: points of movment at start a day
