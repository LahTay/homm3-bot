greedy_algorithm module¶

Script containing functions used determining the most optimal path using greedy alghoritms

greedy_algorithm.greedy(items, maxcal, keyfunction)[source]¶

Function which find the best path from current point to objects in max of movments points. It sorting objects by their value and find path with the biggest sum of their values. :param items: list of objects :param maxcal: max of points movments :param keyfunction: Values of classes to sort it from the most valuable :return: list of objects, sum of value objects in a list of objects
