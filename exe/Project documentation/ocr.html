ocr module¶

Script containing OCR functions that help us with reading text.

ocr.battle_unit_count(img, scale=4, red_unit=False)[source]¶

Used in battle - different thresholds only works for red player The given image has to contain just one text without the picture

Parameters:

  • img – Input image

  • scale – Parameter used in resizing windows.

  • red_unit – Boolean. Set to true if a unit is red

Returns:

text

ocr.prepare_text(image, scale, building=False, dilate=False, red_text=False)[source]¶

Prepare text for tesseract usage - thresholding and dilation

Parameters:

  • image – Input image.

  • scale – Parameter used in resizing windows. Multiplied by height and width

  • building – Boolean. Set to true if we are operating on the building

  • dilate – Boolean. Set to true if dilation operation is to be done.

  • red_text – Boolean. Set to true if we are reading red text.

Returns:

mask put on an image

ocr.read_generic_numbers(img, scale=4, banned_psm=None, redtext=False)[source]¶

Eliminates all answers it finds that contain letters The given image has to contain just one text without the picture

Parameters:

  • img – Input image

  • scale – Parameter used in resizing windows.

  • banned_psm – Which PSM we do not want to use

  • redtext – if background of image is red

Returns:

text, psm.

ocr.read_generic_text(img, scale=4)[source]¶

The given image has to contain just one text without the picture Bigger scale can make letters unreadable because of pixelation especially if they are small at the beginning

Parameters:

  • img – Input image

  • scale – Parameter used in resizing windows

Returns:

text

ocr.read_text(image, psm=10, battle=False, fml=False)[source]¶

It can usually read HOTA font clearly if it’s big enough. Typically shouldn’t be used. read_generic_text should be used for general cases

Parameters:

  • image – Input image

  • psm – Page segmentation. Set to 10 by default.

  • battle – Boolean. Set to true if we are reading text in battle

Returns:

text

ocr.read_text_faster(img, scale=4)[source]¶

Same thing as read_generic_text but fewer checks so used when there can sometimes be no text

Parameters:

  • img – Input image

  • scale – Parameter used in resizing windows.

Returns:

text
