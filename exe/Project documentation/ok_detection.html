ok_detection module¶

Script containing okay button detection, which helps us with handling windows

ok_detection.check_ok()[source]¶

Function responsible for detecting if there is an okay on the screen. Crucial in window handling. :return: 0 - no okay button, 1 - green okay button, 2 - grey okay button, 3 - skill okay button, 4 - city okay button

ok_detection.compare(result, result_image, image)[source]¶

Single function to detect if the best match image is exact same as we need :param result: image of the best match :param result_image: stock image (e.g. battle_result.png) :param image: screenshot :return: returning 1 if its exact the same
