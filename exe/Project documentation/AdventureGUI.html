AdventureGUI.change_artifact(slot_id_from, slot_id_to, additional_id=0)[source]¶

Artifacts are labeled according to https://heroes.thelazy.net/index.php/Artifact you can either pass in the slot number or the slot name additional_id is the number given to miscellaneous or ring artifacts and ones in the backpack

Parameters:

  • slot_id_from – can be either slot number according to the dict or the slot name as str

  • slot_id_to – can be either slot number according to the dict or the slot name as str

  • additional_id – needed when inputting ring, misc or backpack, id of the additional slot
