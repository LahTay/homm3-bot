group_value module¶

group_value.group_value(coords: tuple, our_map, radius=2)[source]¶

Adds additional value to objects next to each other. If the value was < 0 from the beginning then no value is added As its a function that checks the surrounding objects value, that value has to be set already Adding the value immediately to a function would artificially increase the value of the next object that is next to it so the entire map should be, ideally, checked 3 times

Parameters:

  • hero – Our hero

  • coords (tuple) – Coordinates of the tile checked

  • our_map – Our entire found map

  • radius – how many squares around the tile are to be checked

Returns:

Value to be added to the checked tile
